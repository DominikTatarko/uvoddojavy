ÚVOD

Java, od svojho zrodu v roku 1995, zaujíma vývojárov svojou všestrannosťou a schopnosťou ponúkať jednoduché riešenia pre komplexné vývojové úlohy. Vytvorená spoločnosťou Sun Microsystems s cieľom prekonať obmedzenia existujúcich programovacích jazykov, Java sa stala synonymom pre platformovú nezávislosť a bezpečný vývoj.

Java Virtual Machine (JVM) stojí v srdci tohto jazyka, umožňujúc aplikáciám bežať na rôznych operačných systémoch bez straty výkonu. Táto jedinečná vlastnosť uvoľňuje vývojárov od starostí o hardvérové špecifikácie, čím umožňuje vytvárať aplikácie, ktoré sú prenositeľné a ľahko spraviteľné.

V oblasti objektovo-orientovaného programovania je Java príkladom elegancie a jednoduchosti. Jej syntax je intuitívna a ľahko sa naučí, čo uľahčuje novým vývojárom vstup do sveta programovania. Zároveň však poskytuje pokročilé možnosti pre skúsených programátorov, čím zabezpečuje univerzálnosť pre rôzne úrovne skúseností.

VÝHODY

  1. 1.Platformová Nezávislosť: Java je navrhnutá tak, aby bola schopná bežať na rôznych platformách s pomocou Java Virtual Machine (JVM), čo umožňuje prenositeľnosť kódu bez nutnosti opätovnej kompilácie.
  2. 2.Objektovo-Orientovaný Prístup: Java je plne objektovo-orientovaný jazyk, čo zjednodušuje návrh a implementáciu softvéru, a zároveň podporuje modulárny a udržateľný kód.
  3. 3.Rozsiahla Knižnica a Framework Ekosystém: Java disponuje veľkým množstvom štandardných knižníc a frameworkov, ako napríklad Java Standard Edition (SE) a Java Enterprise Edition (EE), čo zjednodušuje vývoj a zrýchľuje proces tvorby aplikácií.
  4. 4.Bezpečnosť: Jazyk Java má zabudované bezpečnostné mechanizmy, ako napríklad bezpečnostné vrstvy v JVM a systémy riadenia prístupu, ktoré chránia aplikácie pred rôznymi hrozbami.
  5. 5.Široké Použitie: Java nájdeme v rôznych oblastiach vývoja, vrátane mobilného vývoja (Android), serverových aplikácií, webových aplikácií, veľkých podnikových systémov a vložených zariadení.
  1. 6.Multi-Threaded Podpora: Java poskytuje zabudovanú podporu pre viacvláknové programovanie, čo umožňuje efektívne využívať výpočtový výkon moderných viacjadrových systémov.
  2. 7.Automatické Riadenie Pamäte: Java používa automatický garbage collector, čo eliminuje potrebu manuálneho riadenia pamäte a pomáha zabrániť únikom pamäte.
  3. 8.Jednoduchá Naučiteľnosť: Syntax jazyka Java je považovaná za jednoduchú a intuitívnu, čo uľahčuje novým vývojárom vstup do programovania.
  4. 9.Komunitná Podpora: Java má veľkú a aktívnu komunitu vývojárov, ktorá znamená dostupnosť bohatého zdroja dokumentácie, tutoriálov a diskusných fór.
  5. 10.Stabilná a Vyspelá Platforma: Java má dlhú históriu a je dobre etablovaná, čo znamená, že poskytuje stabilnú a vyspelú platformu pre vývoj, s pravidelnými aktualizáciami a podporou zo strany spoločnosti Oracle.

ECLIPSE

Eclipse je výkonné integrované vývojové prostredie (IDE), ktoré podporuje tvorbu a úpravu kódu v rôznych programovacích jazykoch. Jeho univerzálnosť umožňuje prácu s rôznymi technológiami, vrátane Javy, C++, Pythonu a ďalších. Vďaka rozšíriteľnosti a pluginom je prispôsobiteľné potrebám vývojárov. Eclipse ponúka integrovanú podporu pre verzovacie systémy, čo uľahčuje sledovanie zmien v kóde a spoluprácu v tíme. S nástrojmi na refaktoring a rýchlym vyhľadávaním zlepšuje kvalitu kódu. Prispôsobiteľné rozhranie umožňuje individuálne nastavenie pracovného prostredia. Jeho silné stránky sú viditeľné aj v podpore rôznych technológií, frameworkov a platforiem. Aktívna komunita vývojárov zabezpečuje pravidelné aktualizácie a pridávanie nových funkcií, čím Eclipse zostáva spoľahlivým a výkonným nástrojom pre rozmanité vývojové projekty.

PREMENNÉ A DATOVÉ TYPY

Premenné sú kontajnery na ukladanie údajových hodnôt.

V jazyku Java existujú rôzne typy premenných, napríklad:

  • String- ukladá text, napríklad „Ahoj“. Hodnoty reťazcov sú ohraničené dvojitými úvodzovkami
  • int- ukladá celé čísla (celé čísla) bez desatinných miest, napríklad 123 alebo -123
  • float- ukladá čísla s pohyblivou rádovou čiarkou s desatinnými miestami, napríklad 19,99 alebo -19,99
  • char- ukladá jednotlivé znaky, ako napríklad 'a' alebo 'B'. Hodnoty znakov sú ohraničené jednoduchými úvodzovkami
  • boolean- ukladá hodnoty s dvoma stavmi: true alebo false

SYNTAX

type variableName = value;

Príklad

Vytvorte premennú s názvom name of type Stringa priraďte jej hodnotu „ John “:

String name = "John";
System.out.println(name);

OPERÁTORY

Operátory sa používajú na vykonávanie operácií s premennými a hodnotami.

V nižšie uvedenom príklade používame + operátor na sčítanie dvoch hodnôt:

Príklad

int x = 100 + 50;

Príklad

int sum1 = 100 + 50; // 150 (100 + 50)
int sum2 = sum1 + 250; // 400 (150 + 250)
int sum3 = sum2 + sum2; // 800 (400 + 400)

Java rozdeľuje operátorov do nasledujúcich skupín:

# Skupina Operatory
1 Aritmeticke +, -, *, /, %, ++, --
2 Priraďovacie =, +=, -=, %=
3 Porovnavacie ==, !=, >, <
4 Logicke &&, ||, !

IF-ELSE

Už viete, že Java podporuje obvyklé logické podmienky z matematiky:

  • Menej ako: a < b
  • Menšie alebo rovné: a <= b
  • Väčšie ako: a > b
  • Väčšie alebo rovné: a >= b
  • Rovná sa a == b
  • Nerovná sa: a != b

Tieto podmienky môžete použiť na vykonanie rôznych akcií pre rôzne rozhodnutia.

Java má nasledujúce podmienené príkazy:

  • Používa sa ifna určenie bloku kódu, ktorý sa má vykonať, ak je zadaná podmienka pravdivá
  • Použite elsena určenie bloku kódu, ktorý sa má vykonať, ak je rovnaká podmienka nepravdivá
  • Použite else ifna určenie novej podmienky na testovanie, ak je prvá podmienka nepravdivá
  • Používa sa switchna zadanie mnohých alternatívnych blokov kódu, ktoré sa majú vykonať

Syntax

if (condition) {
// block of code to be executed if the condition is true
}

Príklad

int x = 20;
int y = 18;
if (x > y) {
System.out.println("x is greater than y");
}

FOR CYKLUS

Keď presne viete, koľkokrát chcete prechádzať blokom kódu, použite slučku for:

Syntax

for (statement 1; statement 2; statement 3) { // code block to be executed }

Príklad

Tento príklad vytlačí iba párne hodnoty medzi 0 a 10:

for (int i = 0; i <= 10; i = i + 2){
System.out.print(i+" ");
}

Druhá uloha

public class AritmetickeOperatory {
public static void main(String[] args) {
// Deklarácia a inicializácia premenných typu int
int prvocislo = 20;
int druhocislo = 5;
// Výpočet aritmetických operácií
int sucet = prvocislo + druhocislo;
int rozdiel = prvocislo - druhocislo;
int sucin = prvocislo * druhocislo;
int podiel = prvocislo / druhocislo;
int zvysok = prvocislo % druhocislo;
// Výpis výsledkov do konzoly System.out.println("Súčet: " + sucet);
System.out.println("Rozdiel: " + rozdiel);
System.out.println("Súčin: " + sucin);
System.out.println("Podiel: " + podiel);
System.out.println("Zvyšok po delení: " + zvysok);
} }

Uloha tri

import java.util.Scanner;
import java.util.Calendar;
public class VekovyTest {
public static void main(String[] args) {
// Vytvorenie inštancie Scanner pre načítanie vstupu od užívateľa
Scanner scanner = new Scanner(System.in);
// Načítanie roka narodenia od užívateľa
System.out.println("Zadajte rok narodenia: ");
int rokNarodenia = scanner.nextInt();
// Získanie aktuálneho roka
int aktualnyRok = Calendar.getInstance().get(Calendar.YEAR);
// Výpočet veku
int vek = aktualnyRok - rokNarodenia;
// Testovanie veku a výpis výsledku
if (vek > 18) {
System.out.println("Máte viac ako 18 rokov.");
} else {
System.out.println("Nemáte ešte 18 rokov.");
}
// Zatvorenie scanneru
scanner.close();
} }

Štvrta úloha

public class SucetCisel {
public static void main(String[] args) {
// Inicializácia premennej pre súčet
int sucet = 0;
// Cyklus na sčítanie čísel od 0 do 20
for (int i = 0; i <= 20; i++) {
sucet += i;
}
// Výpis výsledku do konzoly
System.out.println("Súčet všetkých čísel od 0 do 20 je: " + sucet);
} }

Uloha 5

public class SucetASucin {
public static void main(String[] args) {
// Inicializácia Scanneru pre načítanie vstupu od užívateľa
Scanner scanner = new Scanner(System.in);
// Inicializácia premenných pre súčet a súčin
int sucet = 0;
int sucin = 1; // Počiatočná hodnota pre súčin je 1, aby nebola nulová hodnota
// Načítanie 5 celých čísel od užívateľa
for (int i = 1; i <= 5; i++) {
System.out.println("Zadajte celé číslo č. " + i + ": ");
int cislo = scanner.nextInt();
// Sčítanie do súčtu a násobenie do súčinu
sucet += cislo;
sucin *= cislo;
}
// Výpis výsledkov do konzoly
System.out.println("Súčet zadaných čísel je: " + sucet);
System.out.println("Súčin zadaných čísel je: " + sucin);
// Zatvorenie scanneru
scanner.close();
} }
